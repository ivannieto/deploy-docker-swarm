# Local deployment using Vagrant

## Summary

## Prerequisites

- Docker for Mac or Docker for Windows
- VirtualBox
- Vagrant
- Ansible

### Vagrant plugins

- vagrant-host-shell
- vagrant-alpine
- vagrant-vbguest

The setup script will take care of installing the vagrant dependencies.

## Run

```sh
$ sh deploy.sh
```
