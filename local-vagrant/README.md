## Run

Run with default options (1 manager, 2 workers).
```sh
$ sh setup.sh
```

Run with options:
```sh

# -m - Amount of managers (defaults to 1)
# -w - Amount of workers (defaults to 2)
# -s - Memory allocation per machine (defaults to 1024)

$ sh setup.sh -m 2 -w 3 -s 1024
```
